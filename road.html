¿Qué es Literalura?

Literalura es una aplicación de consola construida con Spring Boot que permite buscar y gestionar información sobre libros utilizando la API pública de Gutendex.

🎯 Objetivo: Consumir una API externa, procesar datos JSON y almacenarlos en una base de datos PostgreSQL (próximamente).

Tecnologías Utilizadas

  • Java 17 - Lenguaje de programación
  • Spring Boot 4.0.1 - Framework principal
  • Spring Data JPA - Para persistencia de datos
  • PostgreSQL - Base de datos relacional
  • Jackson - Para parsear JSON
  • Maven - Gestión de dependencias

Funcionalidades Actuales

✅ Buscar libro por título - Consumiendo la API de Gutendex
⏳ Pendiente de implementación:
  • Listar libros registrados
  • Listar autores registrados
  • Listar autores vivos en un año determinado
  • Listar libros por idioma

Componentes Clave

1. LiteraluraApplication.java

@SpringBootApplication public class LiteraluraApplication implements CommandLineRunner { // Implementa CommandLineRunner para ejecutar código al iniciar // Inyecta Principal mediante constructor (inyección de dependencias) }

Función: Punto de entrada de la aplicación. Ejecuta el menú principal al iniciar.

2. Principal.java (@Component)

@Component public class Principal { private final ConsumoAPI consumoAPI = new ConsumoAPI(); private final ConvierteDatos conversor = new ConvierteDatos(); private final Scanner sc = new Scanner(System.in); }

Función: Maneja la interfaz de usuario por consola con un menú interactivo usando colores ANSI.

3. ConsumoAPI.java

public String obtenerDatos(String url) { HttpClient client = HttpClient.newHttpClient(); HttpRequest request = HttpRequest.newBuilder() .uri(URI.create(url)) .GET() .build(); // Retorna el JSON como String }

Función: Realiza peticiones HTTP GET a la API de Gutendex usando el HttpClient de Java 11+.

4. ConvierteDatos.java

public class ConvierteDatos implements IConvierteDatos { private ObjectMapper mapper = new ObjectMapper(); public T obtenerDatos(String json, Class clase) { return mapper.readValue(json, clase); } }

Función: Usa Jackson para convertir JSON a objetos Java de manera genérica.

5. Records de Datos

// Mapean directamente el JSON de la API @JsonIgnoreProperties(ignoreUnknown = true) public record DatosLibro( Integer id, @JsonAlias("title") String titulo, @JsonAlias("authors") List autores, @JsonAlias("languages") List idiomas, @JsonAlias("download_count") Integer cantidadDeDescargas ) {}

Función: Records inmutables que mapean el JSON de Gutendex usando anotaciones de Jackson.

Flujo de Ejecución Actual

1. Inicio
Spring Boot arranca → LiteraluraApplication.run() → principal.iniciar()
2. Menú Principal
Usuario selecciona opción 1 (Buscar libro por título)
3. Input del Usuario
Usuario ingresa: "Don Quijote"
4. Construcción de URL
URL = "https://gutendex.com/books/?search=Don%20Quijote"
5. Petición HTTP
ConsumoAPI.obtenerDatos(url) → GET request → JSON String
6. Conversión JSON
ConvierteDatos.obtenerDatos(json, RespuestaGutendex.class)
Jackson mapea JSON → Objeto RespuestaGutendex
7. Extracción de Datos
Se toma el primer resultado: DatosLibro libro = respuesta.results().get(0)
8. Mostrar en Consola
Se imprime con colores ANSI: Título, Autor, Idioma, Descargas
